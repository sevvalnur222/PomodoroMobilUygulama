Bu projede, dijital dikkat dağınıklığını azaltmayı ve kullanıcının odaklanma verimliliğini ölçmeyi amaçlayan bir Odaklanma Takibi ve Raporlama Uygulaması geliştirilmiştir. Uygulama React Native (Expo) kullanılarak oluşturulmuş olup iki ana ekrandan oluşmaktadır: zamanlayıcı ekranı ve raporlama ekranı. Kullanıcı, odaklanma seansı başlatmadan önce kategori seçimi yapabilmekte; ardından ayarlanabilir bir geri sayım sayacı çalıştırılmaktadır. AppState API sayesinde kullanıcı uygulamadan ayrıldığında sistem bunu “dikkat dağınıklığı” olarak kaydetmekte ve sayacı otomatik olarak duraklatmaktadır. Seans tamamlandığında süre, kategori ve dikkat dağınıklığı sayısı veritabanına kaydedilerek seans özeti oluşturulmaktadır.

Raporlama ekranında, kaydedilen seanslara ait tüm istatistikler kullanıcıya anlamlı görseller eşliğinde sunulmaktadır. Toplam odaklanma süreleri, günlük performans ve dikkat dağınıklığı sayıları özetlenmiş; son 7 güne ait odaklanma süreleri çubuk grafik ile, kategorilere göre dağılım ise pasta grafik ile görselleştirilmiştir. Kod yapısı component-temelli olacak şekilde organize edilmiş, proje sürüm kontrolü için GitHub kullanılmış ve farklı günlerde yapılan commit’lerle geliştirmenin ilerleyişi belgelenmiştir. Bu proje, hem odaklanma süreçlerinin izlenmesini sağlayan işlevsel bir mobil uygulama ortaya koymakta hem de kullanıcı davranışlarını anlamlandıran analitik bir altyapı sunmaktadır.

Proje, Expo altyapısı kullanılarak geliştirilmiştir. Uygulamayı çalıştırmak için bilgisayarda Node.js, mobil cihazda ise Expo Go uygulaması yüklü olmalıdır. Proje GitHub üzerinden indirildikten sonra proje dizininde npm install komutu ile bağımlılıklar yüklenir ve ardından npx expo start komutu çalıştırılır. Açılan QR kod Expo Go uygulaması ile okutularak uygulama gerçek cihaz üzerinde çalıştırılabilir.
